# vue-touchfeedback

> Touch Feedback for Vue.js

## Demo

https://tmiame.github.io/vue-touchfeedback/example/index.html

## Install

```
npm install vue-touchfeedback
```


## Usage

__CommonJS__

```
var VueTouchFeedback = require('vue-touchfeedback')
Vue.use(VueTouchFeedback)
```
